# FPrimeAI Client Platform Site Last Updated: 2026-02-13 13:00— Static Site Starter

This ZIP contains a ready-to-edit, multi-page, client-facing website starter for FPrimeAI.

## What’s included
- Fully static HTML/CSS/JS (no build tools required)
- Pages:
  - `index.html` (Home)
  - `platform.html`
  - `products.html`
  - `solutions.html`
  - `integrations.html`
  - `resources.html`
  - `company.html`
  - `contact.html`
- Assets:
  - `assets/css/styles.css`
  - `assets/js/main.js`
  - `assets/img/dashboard-placeholder.svg`

## How to run (local)
Option A (quick): open `index.html` in your browser.

Option B (recommended): use a local static server so links behave consistently.
Example:
- Python: `python -m http.server 8080`
- Then open: `http://localhost:8080`

## Customize (important)
1) Replace placeholder text with your real positioning and case studies.
2) Replace `assets/img/dashboard-placeholder.svg` with real screenshots.
3) Update demo email recipient:
   - Open `assets/js/main.js`
   - Replace `sales@fprime.ai` with your real inbox.
